# Small Web App

A simple web application with a homepage that allows changing a message.

## How to Run

1. Open the `index.html` file directly in your web browser.

2. To serve it with a local server using Node.js:
   - Run:
     ```
     node server.js
     ```
   - Then open http://localhost:3000 in your browser.

If you have Python installed, you can also run:

```
python -m http.server 8000
```

Then open http://localhost:8000 in your browser.

## Features

- Displays a welcome message.
- Button to change the message dynamically using JavaScript.
